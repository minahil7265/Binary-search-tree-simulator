# Binary-search-tree-simulator
A C++ implementation of Binary Search Tree (BST) with insertion, deletion, traversal, and performance analysis using chrono timing for benchmarking operations.

## 🚀 Features

### 🌿 Binary Search Tree Operations
- Insert nodes
- Search values
- Delete nodes
- Find minimum and maximum values
- Calculate tree height

### 🔄 Tree Traversals
- Inorder traversal
- Preorder traversal
- Postorder traversal

### ⚡ Performance Analysis
- Measures insertion time
- Measures traversal time
- Uses `chrono` for high precision timing
- Runs multiple test sets (5 × 100 random values)
- Displays average execution time

### 📊 Demo Mode
- Predefined dataset
- Step-by-step deletion demonstration
- Tree state visualization after operations
